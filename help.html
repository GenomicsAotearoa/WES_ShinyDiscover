Loading a sample

To load the results of an annotation run enter the sample ID in the side panel of one of the tier tabs. There are 4 public (1000G) exomes present on the server. If these have been annotated their results can be accessed/viewed by entering one of the following IDs:

  • HG00103
  • HG00108
  • HG00259
  • HG00284

Variant filtering / viewing

There are 4 tabs, one for each tier (variants are filtering into these tiers using VCF-DART). Each tier tab has an interactive table which can be easily manipulated, i.e. sorted using the arrows, search/filtered by typing in the boxes at the bottom of each column. Columns can be toggled by clicking the check boxes located in the side panel of the tab.

MutationAssessor

The list of variants present in this tab are extracted from the provided VCF file based on their MutationAssessor flag - so this is a short list of some of the most predicted damaging variants.

GO Term searching

Enter a gene symbol (i.e. DDO) or a comma separated list of gene symbols (i.e. DDO, FTO, NOTCH3) and press the button to annotate with GO terms.

Genome Viewer

Here you are ale to load various tracks and view them alongside each other.

There are currently 4 bam files loaded, one for each of the public (1000G) exomes loaded on the server.

Downloads

Here you are currently able to download the clinical report (Word docx format) and log file, both which are generated during the annotation and filtering stage of the pipeline.
